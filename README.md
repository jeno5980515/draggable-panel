# draggable-panel
A pure JavaScript plugin to set your div draggable. 
## Usage 
```javascript
<div id="mypanel" >
  <div id="mytoggle" >Drag me</div>
</div>
<script>
  DraggablePanel.init('mypanel','mytoggle') ;
<script>
```
